Tiny Mascot

Embeddable pixel mascot engine — sprite sheets, canvas rendering, Shadow DOM isolation.

👆 Mascot is live in the bottom-right corner. Click it to trigger the react animation.

Sprite Sheet Breakdown

The engine reads frames left-to-right. Each frame is frameWidth × frameHeight pixels. Frames wrap to the next row when the image width is exceeded.

Idle animation — frames 0–3, loops continuously
React animation — frames 4–6, plays once on click then returns to idle

metadata.json Format

{
  "frameWidth": 32,        // width of a single frame in pixels
  "frameHeight": 32,       // height of a single frame in pixels
  "animations": {
    "idle": {
      "frames": [0, 1, 2, 3],  // zero-based frame indices, left → right
      "loop": true              // repeats forever
    },
    "react": {
      "frames": [4, 5, 6],       // plays once on click, then idle resumes
      "loop": false
    }
  }
}
📐
Frame position formula: Frame N starts at x = (N % framesPerRow) × frameWidth, y = floor(N / framesPerRow) × frameHeight — where framesPerRow = imageWidth / frameWidth. Single-row sheets always have y = 0.

Building Your Character Sheet

Lay out all animation frames on a single PNG. Frames are uniform — every cell is exactly frameWidth × frameHeight pixels. Arrange idle frames first (they loop), then react frames.

🎨
Recommended tools: Aseprite (export spritesheet + JSON), Piskel (free, browser-based), or any pixel editor that exports a packed spritesheet. Set frameWidth and frameHeight to match the export cell size.

Usage

React

// npm install mascot-plugin
import { Mascot } from 'mascot-plugin/react';

<Mascot
  spritesheet="/mascot.png"
  metadata="/metadata.json"
  size={64}
  fps={8}
  position="bottom-right"
  offsetX={20}
  offsetY={20}
/>

Web Component

// import once (e.g. in main.js)
import 'mascot-plugin/web-component';

<tiny-mascot
  spritesheet="/mascot.png"
  metadata="/metadata.json"
  size="64"
  fps="8"
  position="bottom-right"
  offset-x="20"
  offset-y="20">
</tiny-mascot>

Vanilla JS

import { MascotEngine } from 'mascot-plugin';

const engine = new MascotEngine({
  spritesheet: '/mascot.png',
  metadata: '/metadata.json',
  size: 64,
  fps: 8,
  position: 'bottom-right',
  offsetX: 20,
  offsetY: 20,
});

await engine.start();
// engine.stop() to unmount

Position Presets

position Anchor offsetX (positive =) offsetY (positive =)
top-left Top-left corner → right into viewport ↓ down into viewport
top-right Top-right corner ← left (inward) ↓ down into viewport
bottom-left Bottom-left corner → right into viewport ↑ up (inward)
bottom-right Bottom-right corner ← left (inward) ↑ up (inward)
center Viewport center → right from center ↓ down from center
